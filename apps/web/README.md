# Web App

This directory will contain the ProofBoard web workspace.

Expected surfaces:

- landing page
- project upload screen
- protocol map
- intent board
- invariant board
- assumption debt board
- verification ledger
- generated harness screen
- results screen
- audit packet export

Recommended stack:

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Monaco editor
- React Flow
